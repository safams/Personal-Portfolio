Movie Theatre

This movie theatre booking system was designed to replicate the experience of purchasing a ticket online. Users can choose from multiple movies, timings for a selected movie, and multiple seats from a visual seating chart

This project was made on IntelliJ IDEA using Java and JUnit testing, as well as JSwing to add a visual component.

Recipe Analyzer

The recipe analyzer was made in IntelliJ IDEA and VSCode by expanding on my Java knowledge from the Movie Theater project to add a SQL database using MySQl Workbench. Simply typing in the name of an ingredient or recipe adds it to your database, making it easier than ever to filter out which recipes to make given the ingredients you have. This is an ongoing project.
